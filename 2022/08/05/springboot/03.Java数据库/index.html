  2. 思考

    1. RequestMapping和GetMapping的关系

      1. rM包括getmapping和postmapping
      2. gM和pM易于代码阅读

    2. component注解干嘛用的 //依赖注入、控制反转

      1. 表明将这些类、组件交给Spring管理，不需要程序员手动去new。

      2. 同类

        作用于类上的注解

        @Component、@Responsity、@Service以及@Controller

      3. 扩展

        作用于修饰字段，构造函数、设置方法

        @Autowired、@Resource

    3. controller service mapper 的关系

      1. controller：入口
      2. service：服务主体
      3. mapper：操作数据方法
